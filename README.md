Beautiful changelog and release notes block for WordPress. Display your version history with style.

== Description ==

**JE Changelog Block** is a modern Gutenberg block that helps you create beautiful changelogs and release notes for your products, plugins, themes, or SaaS applications.

Perfect for:

* **SaaS Applications** - Keep users informed about updates
* **WordPress Plugins/Themes** - Document your releases professionally
* **Software Products** - Share version history with customers
* **Mobile Apps** - Display app update notes
* **Documentation Sites** - Maintain organized release notes

### Features

* **Gutenberg Native** - Built specifically for the block editor
* **Timeline Layout** - Clean and modern design
* **5 Change Types** - New, Fix, Improved, Breaking, Deprecated
* **Rich Text** - Add links, bold, italic, code in descriptions
* **Fully Responsive** - Looks great on all devices
* **Lightweight** - No bloat, fast loading

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/je-changelog-block`, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Create or edit a page/post and search for "Changelog" block.
4. Add the Changelog block to your page and start adding entries.

== Development ==

Source code and build instructions are available on GitHub:
https://github.com/jeplugins/je-changelog-block

To build from source:
1. Clone the repository
2. Run `yarn install`
3. Run `yarn build`
